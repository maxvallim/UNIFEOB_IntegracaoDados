Atividade Avaliada de MongoDB – Criação e Manipulação de Banco de Dados

Descrição do Projeto

Você foi contratado para projetar e implementar o banco de dados de um comércio eletrônico especializado na venda de bicicletas e acessórios. O sistema deve ser capaz de armazenar informações relacionadas aos produtos, clientes, pedidos e fornecedores.

Tarefas

  1. Criação do Banco de Dados:

    Configure um banco de dados MongoDB chamado comercio_bicicletas.

  2. Coleções Necessárias:

    Crie as seguintes coleções, com as respectivas estruturas sugeridas (campos e exemplos de dados):

    • produtos: Informações como nome, descrição, preço, categoria (ex.: "bicicleta", "acessório"), estoque e ID do fornecedor.
    • clientes: Detalhes como nome completo, email, endereço, telefone e histórico de compras.
    • pedidos: Contém ID do cliente, lista de produtos (com quantidade), data do pedido, status do pedido (ex.: "pendente", "enviado").
    • fornecedores: Dados do fornecedor, como nome, contato, localização e lista de produtos fornecidos.
  3. Consultas:

    Desenvolva e execute as seguintes consultas:

    • Listar todos os produtos disponíveis de uma determinada categoria.
    • Buscar o histórico de compras de um cliente específico.
    • Atualizar o status de um pedido para "enviado".
    • Listar os produtos com estoque abaixo de um determinado valor.
  4. Inserções e Atualizações:

    Insira pelo menos 5 documentos em cada coleção para simular os dados do comércio eletrônico. Atualize o preço de um produto em uma das categorias.

  5. Relatório:

    Apresente capturas de tela do terminal mostrando as consultas realizadas e os resultados.

Critérios de Avaliação

  • Estrutura do banco de dados adequada às necessidades descritas.
  • Correção das consultas e manipulação de dados.
  • Organização e clareza na apresentação do relatório.
