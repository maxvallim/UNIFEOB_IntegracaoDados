Você foi contratado para criar um pequeno modelo de data warehouse utilizando o MySQL.

O objetivo é armazenar e analisar dados de vendas de produtos. Para isso, você deve seguir os passos abaixo:

Criação do Data Warehouse:

  • Crie um banco de dados chamado datawarehouse.
  • Crie as tabelas de dimensões DimProduto e DimCliente.
  • Crie a tabela de fatos FatoVendas.

Inserção de Dados:

Insira os dados fornecidos (arquivos .csv) nas tabelas DimProduto, DimCliente e FatoVendas.

Criação de uma Procedure de Limpeza:

Crie uma procedure chamada LimparDados que automatize o processo de limpeza dos dados. A procedure deve:

  • Remover espaços no início e no final dos campos.
  • Remover registros duplicados.
  • Remover registros com valores NULL.
  • Normalizar a formatação das datas.

Criação de Views:

Crie 10 views para analisar os dados do data warehouse (os campos exibidos, devem ser suficientes para dar sentido a consulta). As views devem incluir:

  • Produtos Vendidos
  • Vendas por Cliente
  • Vendas por Data
  • Produtos por Categoria
  • Vendas por Cidade
  • Vendas por Mês
  • Clientes por Cidade
  • Produtos por Cliente
  • Vendas por Produto e Cidade
  • Vendas por Produto e Mês

Consultas Avançadas:

Escreva 5 consultas SQL avançadas para obter informações específicas das views criadas. Os campos exibidos, devem ser suficientes para dar sentido a consulta. As consultas devem incluir:

  • Produtos mais vendidos.
  • Clientes que mais compraram.
  • Vendas por mês.
  • Quantidade de produtos por categoria.
  • Vendas por cidade.
