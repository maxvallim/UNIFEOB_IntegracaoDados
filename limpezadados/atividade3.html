Atividade Avaliada: Limpeza de Dados em MySQL

Esta atividade avaliada tem como objetivo praticar a limpeza de dados em MySQL. Você deverá criar scripts de limpeza para as três tabelas fornecidas abaixo.

Criação das Tabelas e Inserção de Dados

CREATE DATABASE atividade_limpeza;
USE atividade_limpeza;

CREATE TABLE alunos (
    id INT PRIMARY KEY,
    nome VARCHAR(100),
    email VARCHAR(100),
    telefone VARCHAR(20),
    endereco VARCHAR(200)
);

CREATE TABLE cursos (
    id INT PRIMARY KEY,
    nome VARCHAR(100),
    descricao TEXT,
    duracao INT
);

CREATE TABLE matriculas (
    id INT PRIMARY KEY,
    aluno_id INT,
    curso_id INT,
    data_matricula VARCHAR(20),
    FOREIGN KEY (aluno_id) REFERENCES alunos(id),
    FOREIGN KEY (curso_id) REFERENCES cursos(id)
);
